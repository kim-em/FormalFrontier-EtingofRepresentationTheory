## Accomplished

### Issue #2135: Prove MoritaStructural sorry (partial)

Restructured the single sorry in `exists_surjection_with_trivial_kernel_head` (MoritaStructural.lean) into a structured proof with 4 steps:

1. **Step 1 (sorry)**: Construct surjection `g : F(B₁) →ₗ[B₂] B₂/JB` with `ker g = J₂·F(B₁)`. This is the head isomorphism — both heads have each simple module with multiplicity 1 (basic algebra hypothesis + equivalence bijects simples).

2. **Step 2 (proved)**: Lift `g` to `f : F(B₁) → B₂` using `Module.projective_lifting_property` (F(B₁) is projective since equivalences preserve projectives).

3. **Step 3 (proved)**: `f` is surjective by Nakayama/nilpotency (`projective_lift_surjective`).

4. **Step 4 (proved)**: `ker f ≤ J₂ • ker f` via splitting: construct section `s` with `f ∘ s = id`, then kernel projection `proj = id - s ∘ f` maps `J₂·⊤` into `J₂·ker(f)`.

Also added `equiv_hom_to_simple_nonzero`: for any simple B₂-module S, there exists a nonzero morphism F(B₁) → S (via adjunction + nontriviality of G(S)).

### Technical challenges resolved

- **HasQuotient synthesis**: `↑P ⧸ JP` where P is a ModuleCat object fails Lean's instance search. Worked around by using `B₂ ⧸ JB` (which works) and expressing the sorry as existence of a surjection rather than an explicit quotient isomorphism.
- **Exists.casesOn into Type**: `obtain ⟨g, ...⟩` from `∃ g, ...` fails when target is `Σ'`. Fixed by using `let g := sorry; have hg := sorry` pattern.
