## Accomplished

Worked on issue #1354: prove GL₂(𝔽_q) conjugacy class cardinalities.

1. **Proved `two_mul_card_nonsquare`**: In a finite field of odd characteristic, 2·|nonsquares| = q-1. Uses the quadratic character sum approach: partition F into {0} ∪ NZSq ∪ NSq, split ∑χ(a)=0 to get |NZSq|=|NSq|, combined with |NZSq|+|NSq|=q-1.

2. **Proved `card_elliptic_fiber`**: For fixed (a,b,c) with c≠0, the count of d with det≠0 and ¬IsSquare(disc) equals |nonsquares|. Uses affine bijection d ↦ (a-b)²+4cd.

3. **Proved `GL2.card_GL2`**: |GL₂(𝔽_q)| = (q²-1)(q²-q), using Mathlib's `Matrix.card_GL_field`.

4. **Proved `GL2.card_partition`**: The four conjugacy class types partition GL₂(𝔽_q).

5. **Proved `GL2.card_isSplitSemisimple`**: |split semisimple| = (q-1)(q-2)q(q+1)/2, via complement from the partition identity. Uses zify+nlinarith for the polynomial identity.

6. **Proved supporting lemmas**: `card_ge_three`, `ringChar_ne_two`, `det_ne_zero`, `isSquare_disc_of_g01_zero`.

7. **Filed sub-issue #1417** for the remaining `card_isElliptic` sorry.
