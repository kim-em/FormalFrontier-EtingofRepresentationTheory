## Accomplished

Proved `pd_le_of_polynomial_gldim` in `Example9_4_4.lean` — the key inductive step for the lower bound of the Hilbert syzygy theorem (issue #1847). This was the last missing piece: if gldim(R[X]) ≤ d+1 then pd_R(M) ≤ d for every R-module M.

The proof combines three ingredients:

1. **SES construction** (`polynomialExtensionSES`): 0 → F(M) →^{X·} F(M) → Q → 0 where F = extendScalars C. Mono proved by constructing divX ⊗ id as left inverse via `TensorProduct.lift`.

2. **X-action vanishing** (`ext_eq_zero_of_X_action_vanishing`): From the contravariant LES of Ext, precomposition with X-multiplication is surjective on Ext^{d+1}(F(M), Y₀) (since Ext^{d+2}(Q, Y₀) = 0 by gldim). By `smul_eq_comp_mk₀`, this equals scalar multiplication by X, which is zero when Y₀ has trivial X-action.

3. **Shapiro transfer** (`ext_subsingleton_of_polynomial_trivial_action`): Transfers Ext vanishing from R[X]-modules with trivial X-action to all R-modules via the extension-restriction adjunction. Uses `Module.AEval' 0` to construct the trivial-action module, flatness of R[X] over R for PreservesHomology, and a LinearEquiv bridging the type diamond between `G(N₀)` and `N` via `Module.AEval.C_smul`.

Also proved supporting results:
- `ext_eq_zero_of_X_action_vanishing`: LES + X-action vanishing lemma
- `polynomial_X_mul_mono_extendScalars`: X-multiplication is mono on tensor product
- `hasHomologicalDimensionLE_of_polynomial_succ`: wrapper for the inductive step
