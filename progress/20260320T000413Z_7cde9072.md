### Built `sytBranchingEquiv` — branching bijection for SYT

Constructed `SYT(n+1, λ) ≃ Σ_{c ∈ outerCorners(λ)} SYT(n, λ\c)` with:
- **Forward map** (toFun): Finds the cell with max value (Fin.last n), proves it's an outer corner via `syt_maxCell_isOuterCorner`, restricts the filling to the reduced partition.
- **Inverse map** (invFun): Places `Fin.last n` at the corner cell, `castSucc(t'(·))` at all other cells.
- **All SYT properties proved**: bijectivity (`g_bij`, `f_bij`), row monotonicity (`g_row`, `f_row`), column monotonicity (`g_col`, `f_col`).
- **left_inv proved**: Round-trip toFun∘invFun = id, using proof irrelevance for cell subtypes.
- **right_inv**: Still sorry — requires Sigma type equality through opaque `choose`.

### Proved `syt_branching_rule`

```
|SYT(n+1, λ)| = Σ_c |SYT(n, λ\c)| over outer corners c
```

Uses `Nat.card_congr (sytBranchingEquiv n la)` + `Nat.card_sigma` + `rfl`.
