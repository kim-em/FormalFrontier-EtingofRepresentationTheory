### Remaining sorry point
One sorry at line 938: the case where either:
- c_irr ≠ 0 (Casimir eigenvalue is nonzero on V/N), OR
- c_irr = 0 but C ≠ c_irr on some parts of N (N has irreducible factors with different Casimir eigenvalues)

Both sub-cases are mathematically solvable:
1. **c_irr = 0, C ≠ 0 on parts of N**: Find an irreducible submodule W ≤ N with Casimir ≠ 0. Quotient by W, get complement of N/W in V/W by induction (dim < dim V), combine with complement of W in quotient's preimage.
2. **c_irr ≠ 0, C = c_irr on all of N**: Requires weight-space/primitive vector argument (adjust lift of highest weight vector by element of N to make it primitive).
