### New file: `Infrastructure/SpechtModuleSimple.lean`

Three main results, all sorry-free:

1. **`YoungSymmetrizerZ_sandwich_basis`**: For each σ, the element c_ℤ * of(σ) * c_ℤ is proportional to c_ℤ in ℤ[S_n]. Derived from the ℂ sandwich property (Lemma 5.13.1) via injectivity of ℤ → ℂ.

2. **`YoungSymmetrizerK_sandwich_ℚ`**: The full sandwich property c * x * c = f(x) • c over ℚ, obtained by base change from ℤ and extending by linearity.

3. **`SpechtModuleK_isSimpleModule`**: The Specht module V_λ = ℚ[S_n]·c_λ is a simple left ℚ[S_n]-module. Uses Maschke's theorem (semisimplicity) + sandwich property + c² = αc with α ≠ 0.

### Approach

Rather than reproving the combinatorial infrastructure (row/column absorption, pigeonhole transposition) over ℚ, the proof derives the ℤ sandwich identity from the existing ℂ proof using the base change through ℤ:
- Over ℂ: sandwich holds (Lemma 5.13.1)
- The proportionality constant is an integer (evaluate at identity, use c_ℤ(1) = 1)
- By injectivity of ℤ → ℂ: sandwich holds over ℤ
- Map to ℚ (or any commutative ring)
