### `symGroupImage_isSemisimpleRing`
The image of k[Sₙ] in End(V⊗ⁿ) is semisimple (for CharZero fields).
- Built `symGroupMonoidHom`: monoid hom `Perm(Fin n) →* End(V⊗ⁿ)` via `PiTensorProduct.reindex`
- Built `symGroupAlgHom`: algebra hom `k[Sₙ] →ₐ[k] End(V⊗ⁿ)` via `MonoidAlgebra.lift`
- Proved `symGroupAlgHom_range`: range equals `symGroupImage` (adjoin of generators)
- Applied Maschke's theorem (`IsSemisimpleRing k[Sₙ]`) + `RingHom.isSemisimpleRing_of_surjective`

### `symGroupImage_faithfulSMul`
V⊗ⁿ is faithful over symGroupImage. Trivially true since symGroupImage is a subalgebra of End(V⊗ⁿ) — the action is evaluation, so injectivity of the subtype coercion gives faithfulness. The `hN : n ≤ finrank k V` hypothesis is unused (could be removed from the statement).

### Key patterns
- `MonoidAlgebra.lift` is the universal property for lifting monoid homs to algebra homs from group algebras
- `Finsupp.induction_linear` (cases: `zero`, `add`, `single`) is easier to use than `Finsupp.induction`
- `PiTensorProduct.reindex_reindex` handles composition of reindexing
