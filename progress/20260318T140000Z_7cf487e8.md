  - LHS: each summand δ_i([b₁,b₂]) = [δ_i(b₁), δ_i(b₂)] via multilinearity + multiplicativity

  - Key lemmas: δ_mul (multiplicativity), δ_sub (additivity), comm (cross-term commutativity via `PiTensorProduct.map_mul`)
