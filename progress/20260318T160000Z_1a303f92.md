## Accomplished

Completed issue #1060: Proved Gabriel's theorem finiteness (Theorem 6.5.2a) sorry-free.

### Approach

Replaced the original plan (bounding positive root coordinates by `n`) with a cleaner
proof via C.mulVec injection:

1. **cartan_mulVec_injective**: Positive definiteness of C = 2I - adj implies C.mulVec
   is injective (if C·x = C·y then (x-y)ᵀC(x-y) = 0, contradicting pos def).

2. **dotProduct_mulVec_comm**: Bilinear form symmetry: y ⬝ᵥ C *ᵥ x = x ⬝ᵥ C *ᵥ y
   when C is symmetric. Proved by unfolding sums and using C symmetry.

3. **bilinear_expand_sub/add**: B(x±y,x±y) = B(x,x) ± 2B(x,y) + B(y,y) using
   Mathlib's mulVec_sub/add, sub_dotProduct, dotProduct_sub/add + linarith.

4. **cartan_mulVec_bounded**: For positive root d, (Cd)_i ∈ [-2, 2] for all i.
   Uses B(d+e_i, d+e_i) ≥ 2 and B(d-e_i, d-e_i) ≥ 2 (from posdef_min_value).

5. **Theorem_6_5_2a_finiteness**: Positive roots inject via C.mulVec into the
   finite set [-2,2]ⁿ, hence are finite. Uses Set.finite_Icc and
   Set.Finite.of_finite_image.

### Key insight

The original approach (d_i ≤ n) requires spectral theory or Dynkin classification —
neither available. The C.mulVec injection approach avoids coordinate bounds entirely
and yields a cleaner proof.
