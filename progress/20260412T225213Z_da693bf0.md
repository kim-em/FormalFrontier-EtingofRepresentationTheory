### Analysis Details

**D̃₅ proof pattern** (lines 1569-1913) relies on:
1. Two center vertices (2, 3), each receiving exactly 2 arms via disjoint embeddings (starEmbed1 + starEmbed2)
2. Core lemma: if embed1(x) + embed2(y) = 0 then x = 0 ∧ y = 0 (disjointness)
3. Gamma coupling between the two centers creates leaf equalities
4. Leaf equalities + nilpotent shift → N-invariance → nilpotent_invariant_compl_trivial

**Why it fails for Ẽ₆v2** (T_{2,2,2} mixed orientation):
- Center (v0, 3 blocks A,B,C) receives: arm1 via embed2to3_AB (fills A,B) and arm3 via embed2to3_CA (fills A,C)
- **Block A overlap**: embed2to3_AB and embed2to3_CA share block A, so their sum being zero does NOT force both to be zero
- Counterexample: embed2to3_AB((-v,0)) + embed2to3_CA((0,v)) = 0 with nonzero inputs
- The "core decomposition" lemma is mathematically false for this pair

**Why it fails for Ẽ₇** (T_{1,3,3}, all-sink orientation):
- Single center (v0, 4 blocks A,B,C,D) receives 3 arms
- Arm 1: fills A,B,D (not C). Arm 2: fills A,B,C (not D). Arm 3: fills A,C,D (not B)
- No pair of arms has disjoint images at center — each pair shares 2 blocks
- No intermediate vertex has 2+ incoming arms (all intermediates have exactly 1 incoming arrow)
- Subgraph reduction impossible: Ẽ₇ has only 1 vertex of degree 3, but D̃₅ has 2

**Why it fails for T(1,2,5)** (similar to Ẽ₇):
- Single center (v0, 6 blocks) receives 3 arms with overlapping block coverage
- Same structural issue as Ẽ₇

### What proof approach IS needed

These proofs require a fundamentally different strategy than D̃₅:
1. **Block projection analysis**: Use the fact that each block at center is "missing" from exactly one arm. E.g., for Ẽ₇, block B only comes from arms 1&2 (not arm 3), block C from arms 2&3, block D from arms 1&3.
2. **Multi-arm containment chains**: Instead of 2-arm disjointness, use 3-arm constraints simultaneously
3. **Possible approach**: Work with the full 4-block (or 6-block) structure directly, extracting per-block projections and using complement arguments across multiple blocks

Estimated difficulty: Each proof is likely 400-600 lines with novel arguments (not pattern-matching from D̃₅). The nilpotent endgame (nilpotent_invariant_compl_trivial) still applies once N-invariance is established.
