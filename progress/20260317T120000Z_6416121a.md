## Current frontier

The proof of `IsSimpleGroup.no_prime_power_conjClass` requires:
1. Column orthogonality of the character table (summing over irreps, not over group elements)
2. Regular representation decomposition into irreducibles
3. The identity: for g ≠ 1, Σ_V dim(V)·χ_V(g) = 0
4. Algebraic integer argument (Theorem 5.4.4, which depends on Lemma 5.4.5)
5. Scalar action → normal subgroup kernel argument

Mathlib has row orthogonality (`FDRep.char_orthonormal`) but NOT column orthogonality.
