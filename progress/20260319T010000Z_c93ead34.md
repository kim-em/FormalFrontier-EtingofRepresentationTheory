## Accomplished

Proved Theorem 6.8.1 (reaching simple roots via reflections) — issue #1015. This is the key technical step toward Gabriel's theorem. The proof is fully sorry-free.

**Proof structure:**
- Strong induction on ∑ dᵢ (sum of coordinates of the dimension vector)
- Base case: ∑ dᵢ = 1 implies d is a simple root (standard basis vector)
- Inductive step: find vertex k with 0 < (Ad)_k ≤ d_k, apply simple reflection s_k, recurse

**Key lemmas proved:**
1. `cartanMatrix_isSymm` — Cartan matrix inherits symmetry from adjacency matrix
2. `simpleReflection_apply_self` — s_i(v)_i = v_i - (Av)_i for symmetric A
3. `simpleReflection_sum` — ∑ s_i(v)_j = ∑ v_j - (Av)_i
4. `simpleReflection_preserves_B` — B(s_i(v), s_i(v)) = B(v,v) via bilinearity and A_{ii} = 2
5. `simpleReflection_nonneg` — s_k(d) ≥ 0 when (Ad)_k ≤ d_k
6. `simpleReflection_nonzero` — s_k(v) ≠ 0 when B(v,v) = 2
7. `sum_one_is_simpleRoot` — non-negative integers summing to 1 form a standard basis vector
8. **`exists_good_vertex`** — the key combinatorial lemma, proved by contradiction:
   - If no good vertex exists, pick k₀ with d_{k₀} > 0 and (Ad)_{k₀} > 0
   - Then (Ad)_{k₀} ≥ 2 by the assumption gap
   - d' = d - e_{k₀} satisfies B(d',d') = 4 - 2·(Ad)_{k₀} ≤ 0
   - If d' ≠ 0: contradicts positive definiteness; if d' = 0: contradicts ∑ d ≥ 2
