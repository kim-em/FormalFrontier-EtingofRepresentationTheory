### Proof strategy (T_col_inc, ~130 lines)

For cells (r₁, col) and (r₂, col) with r₁ < r₂, show T_fun(r₁,col) < T_fun(r₂,col) where T_fun assigns the c-th smallest entry of each row.

1. For each of the w₂ sorted entries of rowEntries r₂ (B), trace back to the source position qi in row r₂
2. Find the position pi in row r₁ at the same column (valid because parts are sorted descending)
3. Column-standardness gives σ⁻¹(pi) < σ⁻¹(qi), providing w₂ distinct elements of rowEntries r₁ (A) each strictly less than the corresponding sorted entry of B
4. For i ≤ col, these entries are all < B.orderEmbOfFin(col) by monotonicity of orderEmbOfFin
5. Counting argument: col+1 distinct elements of A below B.orderEmbOfFin(col) forces A.orderEmbOfFin(col) < B.orderEmbOfFin(col)

### Key helper used
`parts_descending`: proved that sorted parts of a partition are in descending order (via `List.pairwise_iff_get`).
