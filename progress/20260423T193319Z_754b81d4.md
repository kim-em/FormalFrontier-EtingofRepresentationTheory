**Proof structure** (whole-theorem, no new helpers needed):

1. Start from `linearIndependent_iff'` form: given a finite support `s`,
   coefficients `g : _ тЖТ тДЪ`, and `тИС ╬╗ тИИ s, g ╬╗ тАв schurPoly N ╬╗.val = 0`,
   show `g ╬╝ = 0` for every `╬╝ тИИ s`.
2. Multiply the dependence by the Vandermonde determinant
   `╬Ф = det(alternantMatrix N (vandermondeExps N))`. Using
   `schurPoly_mul_vandermonde` each term becomes
   `g ╬╗ тАв det(alternantMatrix N (shiftedExps N ╬╗.val))`.
3. Take the `MvPolynomial.coeff` at
   `Finsupp.equivFunOnFinite.symm (shiftedExps N ╬╝.val)`.
4. By `alternant_coeff_kronecker` (already in the project for
   `schurPoly_injective`), the inner coefficient collapses to
   `[shiftedExps N ╬╗ = shiftedExps N ╬╝]`.
5. `shiftedExps_injective` converts this to `[╬╗ = ╬╝]`, and
   `Finset.sum_ite_eq'` isolates the `╬╝` term: the whole coefficient
   equals `g ╬╝`, which must be 0.

(`schurPoly_mul_vandermonde`, `alternant_coeff_kronecker`,
