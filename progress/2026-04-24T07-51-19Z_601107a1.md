## Accomplished

Partial PR for #2527 (and supersedes #2510): landed **Part A** of the

New code (~240 lines):

- `matchingPerm` (private noncomputable def, induction on `n`): given
  `f g : Fin n → α` with the same multiset image, produces an explicit
  `σ : Equiv.Perm (Fin n)` with `g = f ∘ σ`. Built via `Fin.cases` and
  `Fin.succAbove`, with bijectivity from injectivity (`Fin.succAbove_ne` +
  `σ'.injective`).
- `toMultiset_sum_single_fn` (private): `Finsupp.toMultiset (∑_l single (g l) 1) =
  Multiset.map g univ.val` via `Finsupp.toMultiset_sum`, `toMultiset_single`,
  and Finset induction with `ndinsert_of_notMem`.
- `symTensor_eq_of_multiset_eq`: `symTensor f = symTensor g` when their
  multisets agree, via `matchingPerm` + `symTensor_comp_perm`.
- `polyToTensor_prod_X`: `polyToTensor (∏_l X(f l)) = symTensor f`. Uses
  `prod_X_eq_monomial_fn`, `Module.Basis.constr_basis` on `basisMonomials`,
  `Finsupp.card_toMultiset`, `multisetToTensor` unfolding, and
  `symTensor_eq_of_multiset_eq`.
- `polyRightTransl_isHomogeneous`: `polyRightTransl g` preserves
  `IsHomogeneous m` via `MvPolynomial.IsHomogeneous.aeval` (each generator
  maps to a degree-1 polynomial).
- `polyToTensor_rightTransl_of_isHomogeneous`: the **main equivariance theorem**.
  For `p` homogeneous of degree `n`,
  `polyToTensor (polyRightTransl g p) = tgtGLAction g (polyToTensor p)`.
  Proof: expand `p` as sum of monomials (homogeneous → each `s` has weight `n`),
  pull scalar via `map_smul`, rewrite monomial as ∏ X(canonicalSeq), apply
  `polyRightTransl_prod` + `polyToTensor_prod_X` + `tgtGLAction_symTensor`.
- `homogeneousPolyToTensor_equivariant`: submodule-level corollary, packaged
  with the `polyRightTransl_isHomogeneous` membership witness.

Build: `lake build EtingofRepresentationTheory.Chapter5.PolynomialTensorBridge`
clean (8031 jobs, only pre-existing CharZero-unused-section linter warnings).
