### Proof structure

1. **embed_sum_zero**: If `starEmbed1 m x + starEmbed2 m y = 0` then `x = 0 ∧ y = 0` (disjointness of the two embedding images). Uses `split_ifs` with `dif_pos`/`dif_neg` for dependent branching.

2. **core decomposition**: If `embed1(x) + embed2(z) ∈ W(center)` and W, W' are complementary invariant submodules, then `x ∈ W(leaf1) ∧ z ∈ W(leaf2)`. Uses `smul_mem (-1 : ℂ)` instead of `neg_mem`/`sub_mem` to avoid instance diamond.

3. **leaf3_sub / leaf4_sub**: Containment propagation from leaf 3 (resp. 4) to leaves 1 and 2.

4. **compl_eq_of_le**: If `A ≤ B`, `A' ≤ B'`, and both pairs are complementary, then `A = B`.

5. **N-invariance (hN₁/hN₂)**: The nilpotent shift preserves W₁(1) and W₂(1), using `h12 ▸ h2` to transport membership.
