Proved `finite_lattice_points_with_B_value` in `Lemma6_7_2.lean` (issue #1185), removing the last sorry in the file.

**Proof strategy**: For a Dynkin Cartan matrix C = 2I - adj and B(v,v) = v ⬝ᵥ Cv = K:
1. Positive semi-definiteness gives B(v ± eᵢ, v ± eᵢ) ≥ 0
2. This bounds each |(Cv)ᵢ| ≤ K + 2
3. Since C.mulVec is injective (from positive definiteness), the level set injects into the finite Icc box {w | ∀ i, -(K+2) ≤ wᵢ ≤ K+2}
4. Preimage of a finite set under an injective map is finite

This avoids eigenvalue theory entirely — the key insight is bounding the *image* Cv rather than v directly.
