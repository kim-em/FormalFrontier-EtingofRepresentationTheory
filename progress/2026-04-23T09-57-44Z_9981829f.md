## Accomplished

Meditate session for #2430: investigated whether the current Lean
framework for proving Ẽ₆/Ẽ₇/Ẽ₈/T(p,q,r) infinite representation type is
mathematically sound. Output is a design document at
`progress/indecomposability-framework-investigation.md`.

Concrete findings:

1. Constructed an explicit non-trivial invariant complementary
   decomposition for `etilde7Rep 1`, confirming the Ẽ₇ framework is
   unsound (analog of the refuted Ẽ₆v2 case on #2417). Verified all 7
   edge invariances by hand.
2. Constructed an analogous decomposition for `t125Rep 1`, confirming
   the T(1,2,5) framework is also unsound.
3. Diagnosed the shared fault: the single N-twist on arm 1 only
   covers the `⟨e₀⟩` sub-block of its target block, leaving the `e_m`
   direction free to peel off into a 1-dim summand. Same mechanism as
   Ẽ₆v2.
4. Explained why `d5tildeRep` is structurally different and does
   work: the γ coupling between the two centers is a bijection
   (I − N invertible since N nilpotent), providing a center-to-center
   bridge absent in Ẽ_n and T(p,q,r).
5. Compared with the book: Etingof's proof of "affine ⟹ infinite
   type" (Problem 6.1.5 + 6.1.2) is an orbit-counting / Tits-form
   argument, not an explicit-rep construction. The current Lean path
   is conceptually different and strictly harder.
6. Proposed two recommended paths forward: (A) implement the book's
   orbit-counting argument (requires significant algebraic geometry
   infrastructure); (B) redesign the explicit reps with multi-twist
   coupling or γ-style bridges. Concrete next-issue plan included.
