## Next step
Both remaining sorries require the **same** missing infrastructure:
constructing a `Module (Matrix (Fin d) (Fin d) k) (M i)` structure and showing
that all simple modules over a matrix ring are isomorphic.

Recommended approach (in order of increasing ambition):
1. **Manual Module construction**: Define `SMul Mat (M i)` via
   `B ‚ÄĘ m = (lift of WA.symm(Pi.single l B)) ‚ÄĘ m`, prove module axioms
   using `hc_identity` (for `one_smul`) and `hWA_mul` + `Pi.single_mul` (for `mul_smul`)
2. **Transfer simplicity**: Show `IsSimpleModule Mat (M i)` via
   `LinearMap.isSimpleModule_iff_of_bijective` with the semilinear identity map
3. **Apply IsSimpleRing.isIsotypic**: For `hŌÉ_inj`, use
   `IsSemisimpleRing.exists_linearEquiv_ideal_of_isSimpleModule` + `IsSimpleRing.isIsotypic`
   to get `M i ‚ČÉ‚āó[Mat] M j`, then convert to `‚ČÉ‚āó[A]` using `IsTorsionBySet.mk_smul`
4. **For hrank (i=j)**: Use `jacobson_density` (in Mathlib) + Schur + `IsAlgClosed` to show
   `A ‚Üí End_k(M_i)` surjective, conclude `dim_k(M_i) = d_{ŌÉ(i)}`, then `dim(Im(E‚āĀ‚āĀ)) = 1`

Key challenge: `Pi.single l` is NOT a ring homomorphism (doesn't preserve 1), so
`Module.compHom` cannot be used. The module must be constructed manually with
explicit proofs of all module axioms. This is the main formalization bottleneck.
