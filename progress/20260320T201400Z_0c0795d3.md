## Accomplished

Proved `GL2.card_isElliptic` in `GL2ConjugacyClasses.lean` (issue #1417, PR #1422).

The theorem states |{g in GL2(F_q) | g is elliptic}| = q^2(q-1)^2/2.

**Proof strategy:** Constructed a bijection psi : g -> (g00, g11, g01, disc(g)) from the elliptic filter to F x F x {c != 0} x {nonsquare}. The key insight is that the inverse map (a,b,c,x) -> (a, b, c, (x-(a-b)^2)/(4c)) automatically satisfies det != 0 because det = ((a+b)^2 - x)/4, and (a+b)^2 is a square while x is a nonsquare, so they cannot be equal. The target set has cardinality q * q * (q-1) * NSq = q^2(q-1)^2/2 using 2*NSq = q-1.

This also makes `GL2.card_isSplitSemisimple` sorry-free (it was already proved by complement from the partition identity, depending on card_isElliptic).
