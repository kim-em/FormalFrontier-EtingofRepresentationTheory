## Current frontier

Three sorry'd helpers remain in the Garnir infrastructure:

1. `garnirSet_has_row_pair` (pigeonhole): show the Garnir set has 2 positions in the same row. This requires showing |A|+|B| = λ_{r₁}+1 ≥ 2, hence one of |A|,|B| ≥ 2, giving two positions in the same row.

2. `left_transposition_negates_garnir`: show of(t)·G = -G for transposition t ∈ S_{A∪B}. Requires re-indexing the sum via left multiplication bijection w ↦ t·w on S_{A∪B}, with sign(t·w) = -sign(w).

3. `garnir_identity_expansion` (the main sorry): the full Garnir reduction. From a_λ·G = 0, extract the identity term to get c_λ = -Σ_{w≠id} sign(w)·a_λ·of(w)·b_λ, then show of(σ)·c_λ = -Σ_w sign(w)·of(σ)·a_λ·of(w)·b_λ as a combination of of(τ)·c_λ with fewer column inversions. This requires:
   - Rewriting each term as a sum over P_λ-cosets
   - Proving the column inversion count decreases for each resulting τ
