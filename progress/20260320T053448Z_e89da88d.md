## Accomplished

Working on issue #1354 (Stage 3.2: Ch5 prove GL2 conjugacy class cardinalities). Proved 2 of the 3 deliverables:

### Proved:
1. **`GL2.card_disc_eq_zero`**: The number of GL₂(𝔽_q) elements with discriminant zero equals (q-1)·q². Split into two cases:
   - g₀₁=0: bijection with 𝔽_q* × 𝔽_q via (a,b) ↦ [[a,0],[b,a]], giving (q-1)·q elements
   - g₀₁≠0: bijection with 𝔽_q* × 𝔽_q* × 𝔽_q via (c,s,d) ↦ [[s+d,c],[-d²/c,s-d]], giving (q-1)²·q elements
   - Total: (q-1)q + (q-1)²q = (q-1)q²

2. **`GL2.card_isParabolic`**: The number of parabolic elements equals (q-1)(q²-1). Follows from disc=0 count minus scalar count via `Finset.card_sdiff_add_card_eq_card`.
