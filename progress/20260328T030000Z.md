## Accomplished

Proved `ext_subsingleton_of_extendScalars` in `ShapiroLemma.lean` — the Shapiro lemma for Ext groups (subsingleton direction). This was issue #1867.

### What was proved:
For `f : R →+* S` with `(extendScalars f).PreservesHomology`, if `Ext^i_S(F(M), N)` is subsingleton then `Ext^i_R(M, G(N))` is subsingleton, where F = extendScalars f, G = restrictScalars f.

### Proof approach:
- **i=0**: Ext^0 ≅ Hom via `addEquiv₀`. The adjunction `homEquiv : (F(M) → N) ≃ (M → G(N))` transfers subsingleton from S-homs to R-homs.
- **i=n+1**: Represent `e₁ - e₂` via `extMk_surjective` as a cocycle `g`. Transfer to S-side via `adj.homEquiv.symm` to get cocycle `g'`. Since S-side Ext is subsingleton, `g'` is a coboundary. Transfer back via `adj.homEquiv` to show `g` is a coboundary, hence `e₁ - e₂ = 0`.
