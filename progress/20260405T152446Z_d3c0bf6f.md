Cherry-picked Steps 2-4 from PR #2159 (projective lifting, Nakayama surjectivity,

**After** (1 sorry):

```lean
have head_iso : (Pt ⧸ JP) ≃ₗ[B₂] (B₂ ⧸ JB) := sorry
let g := head_iso.toLinearMap.comp JP.mkQ
-- surjectivity and kernel derived automatically
```

The remaining sorry is the **head isomorphism**: F(B₁)/J·F(B₁) ≅ B₂/J as B₂-modules.
Both are semisimple modules with each simple appearing exactly once (proved by the
adjunction Hom(F(B₁), S) ≅ G(S) and basic algebra dim_k G(S) = 1). The isomorphism
follows from semisimple module classification by multiplicities, which requires either:
- Krull-Schmidt for semisimple modules, or
- Wedderburn-Artin B₂/J ≅ kⁿ + classify kⁿ-modules by component dimensions

Neither is available as a ready-to-use Mathlib lemma.
