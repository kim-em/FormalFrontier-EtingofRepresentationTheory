## Current frontier

The sorry in `pd_le_of_polynomial_gldim` (line ~230) requires:

1. **SES construction**: 0 → R[X]⊗M →^{X·} R[X]⊗M → M₀ → 0 in ModuleCat R[X]
2. **X-action argument**: From the LES, X-multiplication is surjective on Ext^{d+1}(R[X]⊗M, Y). For Y with trivial X-action, X·α = α.comp(mk₀(X·𝟙_Y)) = 0 (by `smul_eq_comp_mk₀`), so Ext^{d+1}(R[X]⊗M, Y₀) = 0.
3. **Shapiro's lemma**: Ext^i_{R[X]}(R[X]⊗M, Y₀) ≅ Ext^i_R(M, Y₀|_R) connects the R[X]-Ext vanishing to R-Ext vanishing.

Key Mathlib API for the proof:
- `Ext.smul_comp`, `Ext.mk₀_id_comp`: show f* = X-action on Ext
- `Ext.smul_eq_comp_mk₀`: X·α = 0 when X·𝟙_Y = 0
- `extendRestrictScalarsAdj Polynomial.C`: the adjunction for Shapiro
- `preservesProjectiveObjects_of_adjunction_of_preservesEpimorphisms`: extendScalars preserves projectives
