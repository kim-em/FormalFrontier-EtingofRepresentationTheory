### Analysis of hσ_surj

Multiple approaches analyzed, all hitting obstacles:

1. **Construct standard module as A-module**: `Fin (d l₀) → k` as simple A-module via quotient map — universe mismatch (`Type u_1` vs `Type uA`)
2. **Jacobson radical characterization**: Need `⋂ ann(simples) ≤ J(A)` (reverse direction of `hann`) — not in Mathlib
3. **Finrank counting**: Need `∑_i (finrank k (M i))² = finrank k (A/J)` — not formalized
4. **Regular representation decomposition**: Show A/J is semisimple as A-module, apply hM_exhaustive to simple summands — requires change-of-rings infrastructure

The cleanest mathematical proof: if l₀ ∉ range σ, lift c(l₀) to a₀ ∈ A, show a₀ annihilates all M_i, show a₀ acts nontrivially on A/J (via a₀ • c(l₀) = c(l₀) ≠ 0), decompose A/J into simple A-submodules, apply hM_exhaustive, get contradiction. Blocked on showing A/J is semisimple as an A-module (need IsSemisimpleRing → IsSemisimpleModule for quotient ring).

## Next step

Prove `hσ_surj` (surjectivity). Best approach: show that A ⧸ Ring.jacobson A, viewed as a left A-module, is semisimple (since J acts as zero and A/J is a semisimple ring). Then any nonzero submodule contains a simple A-submodule, which by hM_exhaustive is isomorphic to some M_i. The contradiction follows from c(l₀) acting nontrivially on A/J but being zero on all M_i.

Alternatively, add a helper lemma: for a semiprimary ring A, `Ring.jacobson A = ⋂_i Module.annihilator A (M_i)` where M_i ranges over all simple A-modules. This is a classical result but not currently in Mathlib.
