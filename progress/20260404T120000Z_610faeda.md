### Issue #2089: Prove Module.Finite transport in MoritaStructural (2→1 sorry)

Eliminated the `Module.Finite B₂ (F.functor.obj (ModuleCat.of B₁ B₁))` sorry in `basic_morita_regular_module_iso` by restructuring the proof strategy:

- **Changed `iso_of_surjection_with_trivial_kernel_head`** from requiring `[IsNoetherianRing B₂] [Module.Finite B₂ P]` to `[IsArtinianRing B₂]`
- **Key insight**: Instead of Nakayama's lemma (which requires `Module.Finite`), use nilpotency of the Jacobson radical directly. For Artinian rings, `IsSemiprimaryRing.isNilpotent` gives `J^n = 0`. If `ker f = J • ker f`, then `ker f ≤ J^k • ⊤` for all k by induction, so `ker f ≤ J^n • ⊤ = 0`.
- **Technical challenge**: `Module (Ideal R) (Submodule R M)` only exists for `CommSemiring R`, so submodule-level algebraic identities like `mul_smul`, `one_smul` are unavailable. Solved via element-level induction using `Submodule.smul_induction_on`.
- **Result**: The `Module.Finite` sorry is gone; `basic_morita_regular_module_iso` now just needs `IsArtinianRing B₂` (from `IsArtinianRing.of_finite k B₂`).
