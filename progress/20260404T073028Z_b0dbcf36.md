## Accomplished

### Issue #2079: iso_of_glWeightSpace_finrank_eq (Theorem5_22_1.lean 1→0 sorry)

Moved `iso_of_glWeightSpace_finrank_eq` from `Theorem5_22_1.lean` to `FormalCharacterIso.lean`, clearing all sorries from the Weyl character formula file.

**Key finding:** The theorem as stated is **false for non-polynomial representations**. Counterexample: for N ≥ 1, `det⁻¹` and `det⁻²` both have all ℕ-valued weight spaces = ⊥ (the hypothesis is vacuously satisfied), but they are not isomorphic. The theorem is correct when restricted to polynomial representations, which is the only case used downstream (in `Proposition5_22_2`).

**What the proof requires** (for the sorry that now lives in FormalCharacterIso.lean):
1. Complete reducibility of polynomial GL_N representations (decomposition as ⊕ L_λ)
2. Linear independence of Schur polynomials
3. Equal multiplicities ⟹ isomorphism

None of these have existing infrastructure in the project or Mathlib. The Schur-Weyl decomposition in `Theorem5_18_4` provides V^⊗n ≅ ⊕ S_i ⊗ L_i as bare k-linear maps (not equivariant), but connecting this to FDRep-level isomorphisms requires additional work.
