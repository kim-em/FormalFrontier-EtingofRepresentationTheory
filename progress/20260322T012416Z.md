**Why it's hard:** The alternating sum ∑_π sign(π) K(sort(λ+ρ-e_π), ν) = 0 for ν > λ requires cancellation between positive and negative terms. The book proves this via a character norm argument:
1. θ_λ = ∑_{ν≥λ} L_{νλ} χ_ν with L_{λλ} = 1
2. ⟨θ_λ, θ_λ⟩ = ∑_ν L²_{νλ} ≥ 1 (by character orthonormality)
3. ⟨θ_λ, θ_λ⟩ ≤ 1 (by Vandermonde orthogonality on polynomial coefficients)
4. Therefore L_{νλ} = 0 for ν ≠ λ
