## Accomplished

Closed issue #2456 (Schur-Weyl sub-issue #1): formalized the tensor-degree
homogeneity bridge from `formalCharacter = schurPoly` in
`EtingofRepresentationTheory/Chapter5/FormalCharacterIso.lean`.

Three new declarations added:

1. `homogeneousComponent_mul_of_isHomogeneous_right` (private) —
   if `ψ` is homogeneous of degree `n`, then
   `homogeneousComponent (k + n) (φ * ψ) = homogeneousComponent k φ * ψ`.
   This is the graded-algebra shift lemma for multiplying by a
   homogeneous polynomial, proved directly from `coeff_mul`.

2. `schurPoly_isHomogeneous` (public) —
   `(schurPoly N lam).IsHomogeneous (∑ i, lam i)`.
   Proof: if some monomial `x^d` in `schurPoly N lam` has
   `d.degree ≠ ∑ lam`, apply the helper to the identity
   `schurPoly * Δ = alt_{λ+δ}` (from `schurPoly_mul_vandermonde`). The
   `(d.degree + dΔ)`-homogeneous component of both sides forces
   `homogeneousComponent d.degree schurPoly * Δ = 0`, hence
   `homogeneousComponent d.degree schurPoly = 0` (cancel nonzero Δ), but
   that contradicts `coeff d schurPoly ≠ 0`.

3. `weight_magnitude_of_formalCharacter_eq_schurPoly` (public) — the
   issue's deliverable: for a `GL_N(k)`-rep with
   `formalCharacter M = schurPoly N lam`, every weight `μ` with
   `finrank (glWeightSpace M μ) > 0` satisfies `∑ μ = ∑ lam`.
   Proof is a three-step cast: `finrank > 0 ⟹ coeff (μ as Finsupp) ≠ 0`
   in the formal character (via `formalCharacter_coeff`), transport to
   `schurPoly` via the character equation, apply
   `schurPoly_isHomogeneous`, then convert `Finsupp.weight 1 d = ∑ lam`
   into `d.degree = ∑ lam` and use `Finsupp.degree_eq_sum` to read off
   `∑ μ`.
