`∑_{σ∈Sₙ} χ_λ(type(σ)) · χ_{λ'}(type(σ)) = n! · δ_{λ,λ'}`

### Proof chain:
1. **`inv_mul_revPerm_val`**: `((π * Fin.revPerm)⁻¹ i).val = permVandermondeExp N π i` — connects the two forms of Vandermonde exponents via `Fin.revPerm` (reversal permutation)
2. **`shiftedExps_injective`**: `shiftedExps N lam.parts = shiftedExps N lam'.parts ↔ lam = lam'`
3. **`vandermonde_cauchy_permVandermondeExp`**: Reindexes `vandermonde_cauchy_general` from `(π⁻¹ i).val` conditions to `permVandermondeExp` conditions, using double `Fintype.sum_equiv` via `Equiv.mulRight Fin.revPerm` with sign cancellation (`sign(ρ)² = 1`)
4. **`charValue_row_orthogonality`**: Transfers from ℚ to ℂ via `Rat.cast_injective`, applies the Cauchy identity
