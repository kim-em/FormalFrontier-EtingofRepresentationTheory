## Current frontier

The key mathematical blocker is `right_pq_dominance`: proving that `tabloidDominates la (sytPerm n la T) (sytPerm n la T * p * q)` when p ∈ P_λ, q ∈ Q_λ. The existing `column_perm_dominance` proves LEFT multiplication dominance (q⁻¹ * σ), but RIGHT multiplication (σ * p * q) is fundamentally different:

- Left multiplication permutes positions → column-increasing property of SYT applies directly
- Right multiplication permutes entries → need to reason about how row permutations (p) preserve tabloid class and column permutations (q) can only worsen dominance

The proof likely requires: (1) σ_T * p has same tabloid as σ_T (since p ∈ P_λ permutes within rows), then (2) σ_T * p * q is dominated by σ_T * p (column perm worsens dominance), requiring a RIGHT-multiplication version of column_perm_dominance.
