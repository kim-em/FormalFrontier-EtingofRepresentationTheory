### New infrastructure lemmas (all sorry-free):

1. **`leftIdeal_equiv_of_conjugate`**: Conjugate idempotents give A-linearly isomorphic left ideals.
   If `u * e₁ * u⁻¹ = e₂`, then `A·e₁ ≅ A·e₂` via right multiplication by `u⁻¹`.
   This is key infrastructure for Part (ii): showing `A·e_{ij} ≅ A·e_{i1}` for fixed `i`.

2. **`exists_isCoatom_submodule`**: Any nontrivial f.g. module over an artinian ring has a
   maximal (coatom) submodule. Uses Hopkins-Levitzki (f.g. over artinian ⟹ noetherian ⟹ coatomic).

3. **`exists_nonzero_hom_to_simple`**: Any nontrivial f.g. module Q over an artinian ring
   (with exhaustive simple family) has a nonzero Hom to some M_{j₀}. This is the key step
   in Part (iii) that does NOT require uniqueness of projective covers.
