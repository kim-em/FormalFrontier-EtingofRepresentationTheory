### Char 2 issue discovered

The split semisimple and elliptic formulas are **incorrect for p = 2**. In characteristic 2, `disc(g) = (g₀₀-g₁₁)²` is always a square, so `IsElliptic g = False` for all g. But the formula gives nonzero count. Verified by hand for GL₂(𝔽₂): formula says |elliptic| = 2, actual = 0; formula says |split| = 0, actual = 2. The parabolic and scalar formulas ARE correct for all characteristics.

### Proof approach for remaining cardinalities

The cleanest approach uses trace-determinant parametrization:
- For a 2×2 matrix with trace t and determinant d, the number of matrices N(t,d) = q(q-1+r(t,d)) where r is the number of roots of X²-tX+d in 𝔽_q.
- r = 2 if disc = t²-4d is a nonzero square, r = 1 if disc = 0, r = 0 if disc is non-square.
- Count (t,d) pairs for each case to get the class cardinalities.
