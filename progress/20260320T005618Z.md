## Accomplished

Proved 4 key lemmas in `Theorem_Dynkin_classification.lean` for the forward direction of Dynkin classification:

1. **subgraph_contradiction** (hle proof): Completed the sum reindexing argument showing that pushing a null vector from a subgraph into the ambient graph contradicts positive-definiteness. Uses `sum_reindex` helper, `EmbeddingLike.apply_eq_iff_eq`, and `mul_le_mul_of_nonneg_right`.

2. **dynkin_degree_le_three**: Every vertex in a Dynkin diagram has degree ≤ 3. Constructs test vector x(center)=2, x(neighbors)=1, x(else)=0 and shows B(x,x) ≤ 0 if degree ≥ 4, contradicting positive-definiteness. ~80 lines.

3. **adj_sum_eq_degree** (new helper): For a 0-1 adjacency matrix, `∑ b, adj a b = ↑(vertexDegree adj a)`. Connects integer row sum to the natural number vertex degree.

4. **dynkin_has_endpoint**: In a Dynkin diagram with all degrees ≤ 2, there exists a vertex of degree ≤ 1. Proof: if all degrees = 2, the all-ones vector gives B(x,x) = 0, contradicting positive-definiteness.
