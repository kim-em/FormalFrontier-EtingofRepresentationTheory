### Key findings

- The classical proof uses the tabloid-dominance partial order; the
  current framework uses `rowInvCount`, which is a permutation-level
  measure and **does not refine** the tabloid-level dominance order.
  Specifically, a column-standard `τ` with `{τ}` strictly dominating
  `{σ}` can have `rowInvCount(τ) ≥ rowInvCount(σ)` depending on
  intra-row orderings of the same tabloid class.

- Concrete computation on `λ = (2,2)`, `σ = swap(0,1)`: for the one
  non-trivial Garnir permutation `w = swap((1,0), (0,1))`, all four
  `[w·q⁻¹·σ]` (for `q ∈ Q_λ`) collapse to the same tabloid
  `T_1 = {row 0: {1,2}, row 1: {0,3}}`, and the sum of signs over
  `wQ_λw⁻¹ = P_λ` is 0, so `f_w(σ) = 0` trivially. But `T_1` has **no**
  column-standard representative at all, which means `f_w(σ)` cannot in
  general be expressed as `ψ_τ` for a column-standard τ with
  `{τ} = T_1`. The proof must exploit cross-polytabloid cancellation.
