### Issue #2111: Prove non-ADE trees have infinite representation type (partial)

Added two new theorems to InfiniteTypeConstructions.lean (which remains sorry-free):

1. **`star_subgraph_not_finite_type`**: If a graph has a vertex with 4 pairwise non-adjacent neighbors (a K_{1,4} subgraph), then it has infinite representation type. Proof constructs an embedding φ : Fin 5 ↪ Fin n and applies `subgraph_infinite_type_transfer` with `star_not_finite_type`.

2. **`tree_degree_ge_4_not_finite_type`**: A triangle-free graph with a vertex of degree ≥ 4 has infinite representation type. Extracts 4 neighbors using `Finset.exists_subset_card_eq`, proves they form an independent set via the triangle-free hypothesis, then applies (1).

Also added helper definition `vertexDegree` for vertex degree in adjacency matrix graphs.
