## Accomplished

Reduced sorries in `GL2NormalizerInfra.lean` from 4 to 1 (#1517):

### `root_dichotomy_of_deg_two` (was sorry)
Proved: if P(a)=P(b)=P(c)=0 and deg P = 2 with a,b distinct units, then c=a or c=b.
Approach: coprimality of (X-a) and (X-b) via `isCoprime_X_sub_C_of_isUnit_sub`, then
`IsCoprime.mul_dvd` gives (X-a)(X-b)|P, degree argument gives P = const*(X-a)(X-b),
field arithmetic gives c=a or c=b.

### `exists_nonscalar_elliptic` (was sorry)
Proved: there exists a unit in F_{q²} whose embedding is non-scalar.
Approach: contrapositive — if all embeddings are scalar, then algebraMap is surjective,
so finrank ≤ 1, contradicting finrank = 2.

### `normalizer_mem_dichotomy` (was sorry, hardest)
Proved: N(K) = K ∪ σK — every normalizer element is either in K or in the Frobenius coset.
Key sub-result: α₀ ≠ α₀^q (non-scalar element is not Frobenius-fixed). Proved via
Galois theory chain: Frobenius-fixed → all-automorphisms-fixed (|Gal|=2 enumeration
via `Nat.card_eq_two_iff'`) → in range(algebraMap) → scalar → contradiction.

### `normalizer_card` (1 sorry remains)
|N(K)| = 2|K|. The proof strategy is clear (N = K ⊔ σK disjoint, |σK|=|K|) but
the Finset-based cardinality proof had elaboration issues (Finset.image + fin_cases
simplification + DecidableEq synthesis). Left as sorry.
