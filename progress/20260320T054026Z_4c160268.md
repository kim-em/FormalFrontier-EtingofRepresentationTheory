### Restructured alternating_kostka_eq_delta proof

Split into cases:
- **la = nu**: Isolates the rev permutation term, uses `spechtMultiplicity_vanishing` for non-rev
  terms and `spechtMultiplicity_diagonal` for the rev term. Two sub-sorries remain:
  1. `finsuppToPartition_toFinsupp` — round-trip: `finsuppToPartition(la.toFinsupp) = la`
  2. `hrest = 0` — proving `finsuppToPartition(la+ρ-e_π) ▷ la` for π ≠ rev (strict dominance
     of sorted rearrangement). This is the key mathematical content.
- **la ≠ nu**: Sorry'd entirely. This case requires genuine cancellation across all permutation
  terms (not just term-by-term vanishing). The book proves this via an inner product/norm
  argument: `||θ_λ||² = 1` from Vandermonde orthogonality forces all off-diagonal coefficients
  to zero.

### Key mathematical insight

The issue description suggested all non-rev terms vanish individually, which is only true when
nu = la. When nu ≠ la, individual non-rev Kostka numbers m(sort(la+ρ-e_π), nu) can be nonzero;
the cancellation is across ALL terms. This makes the la ≠ nu case fundamentally harder.
