combinatorial heart of Chapter 5's straightening theorem. It proves that a
Garnir-twisted polytabloid `f_w(σ)` lies in the span of lower polytabloids in
the row-inversion order.

**Recent fixes:**
- PR #2390 replaced the trivial Garnir set `{p₁, p₂}` with a proper

- PR #2412 added the `w ∉ ColumnSubgroup n la` hypothesis.
- PR #2415 (fix for #2413) added the `w ∉ RowSubgroup n la` hypothesis,
  resolving the circularity: for `w ∈ RowSubgroup`, `f_w(σ) = ψ_σ` exactly
  (via the new `twistedPolytabloid_row_eq`), which would make the original
  theorem statement circular. The fixed statement only covers `w ∉ P_λ ∪ Q_λ`,
  which is the genuinely combinatorial case.

With both exclusions in place, the core sorry at line 694 is now provable
in principle. This is the "classical Garnir" argument in James/Fulton Ch. 7-8.
