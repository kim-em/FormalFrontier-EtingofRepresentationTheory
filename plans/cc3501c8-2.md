The pigeonhole hypothesis is essentially: w maps a column of the
Garnir set into a row that is already "full" under the partition λ, so
no matter how q permutes within columns, the resulting tabloid is
determined by λ and is independent of q.

```lean
/-- **Pigeonhole collapse**: for `w` supported on the Garnir set
`G = garnirSet p₁ p₂` satisfying an overflow condition (to be made
precise by the worker — the natural form is "after conjugation by q,
the preimage of G under w maps into too few rows, forcing the tabloid
of `w · q⁻¹ · σ` to be constant in q"), the alternating sum
`Σ_{q ∈ ColumnSubgroup} sign(q) · [w · q⁻¹ · σ]` (viewed as a
Finsupp over tabloids) vanishes. -/
private theorem garnir_pigeonhole_collapse
    (σ : Equiv.Perm (Fin n)) (hcs : isColumnStandard' n la σ)
    (p₁ p₂ : Fin n)
    (hrow_eq : rowOfPos la.sortedParts p₁.val = rowOfPos la.sortedParts p₂.val)
    (hcol_lt : colOfPos la.sortedParts p₁.val < colOfPos la.sortedParts p₂.val)
    (w : Equiv.Perm (Fin n))
    (hw_supp : ∀ x, x ∉ garnirSet (la := la) p₁ p₂ → w x = x)
    (hw_collapse : /- pigeonhole hypothesis; see below -/) :
    (∑ q ∈ (ColumnSubgroup n la).carrier.toFinset,
      Equiv.Perm.sign q • Finsupp.single (toTabloid n la (w * q⁻¹ * σ)) 1)
      = 0
```

**Pinning down `hw_collapse`** is part of the deliverables. Candidate
forms (the worker picks one and validates on the counter-example):

- **Form 1 (direct)**: `toTabloid n la (w * q⁻¹ * σ)` is the same for
  all `q ∈ ColumnSubgroup n la`. Then the sum is a single tabloid
  multiplied by `Σ_q sign(q) = 0` (since `ColumnSubgroup` contains a
  transposition).
- **Form 2 (structural)**: there exists a row `r` such that
  `|{x ∈ G : rowOfPos la.sortedParts (σ.symm x.val) = r}| > λ.sortedParts[r]`.
  This is the classical pigeonhole; it implies Form 1.

The worker should pick the form that admits the cleanest proof and
verify it on w ∈ {swap(1,2), (0,1,2)} for the λ=(2,2) counter-example.
