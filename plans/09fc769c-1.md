3. **`Theorem5_18_4_GL_rep_decomposition`** (new theorem,
   `Theorem5_18_4.lean:end`) — the downstream payload. Specializes to
   `V = Fin N → k` and produces, for each `n ≤ N`, a list of

   - `ρ i : GL_N →* End k (L_i)` — via `Module.toModuleEnd k (L_i)` ∘
     `glHom`.
   - `FDRep.of (ρ i)` — using the new `Module.Finite k (L_i)` clause.
