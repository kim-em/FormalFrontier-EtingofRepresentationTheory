This is where the **counter-example bites the per-term approaches** —
individual `(w=(0,2,1), q)` terms do **not** each produce an L-member,
but their **grouped sum** does. Grouping is by col-std equivalence:
pairs `(w, q), (w, q')` are grouped when
`garnirColReindex σ w q · (w · q⁻¹ · σ) = garnirColReindex σ w q' · (w · q'⁻¹ · σ)`
(i.e. identical col-std reindex). The sign-sum over each class is the
coefficient of the corresponding ψ_τ in the broadened L.

            (toTabloid n la τ = toTabloid n la σ ∧

**Proof approach** (from progress file sketch):

1. Partition `ColumnSubgroup n la` into cosets of the stabilizer subgroup
   `Stab_w := {q : q · w · q⁻¹ produces same col-std reindex}` — i.e.
   group by the col-std-equivalence class of `garnirColReindex σ w q`.
2. In each coset, use `garnirColReindex_polytabloid_eq` (from sibling A)
   to factor out the sign and express the coset-sum as a ℂ-multiple of
   a single ψ_τ.
3. For each resulting τ, prove the broadened disjunction:
   - **Main case**: τ's tabloid strictly dominates σ's (classical
     Garnir dominance via James/Fulton argument, restricted to the
     no-collapse w). Progress file anchor: `f_{(0,2,1)}(σ)` produces
     `ψ_[0,2,1,3]` with tabloid `[{0,2}|{1,3}]` strict-dominating σ.
   - **Corner case**: same tabloid, smaller rowInv. Arises when the
     coset representative induces a within-row permutation
     decreasing `rowInvCount'`. Progress file anchor: same `f_{(0,2,1)}`
     produces `−ψ_id` with rowInv 0 < 1.
4. Conclude span membership via `Submodule.sum_mem` + `smul_mem`.
