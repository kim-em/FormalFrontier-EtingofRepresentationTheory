## Deliverables

Add a single theorem of approximately the form:

```lean
/-- Any finite-dimensional polynomial GL_N-representation `M` whose
weights are all of degree `n` decomposes as a direct sum of Schur
modules `SchurModule k N λ` indexed by antitone partitions `λ` of `n`
with at most `N` parts. -/
theorem decompose_polynomial_gl_rep
    {k : Type*} [Field k] [CharZero k] [IsAlgClosed k]
    {N n : ℕ}
    (M : FDRep k (Matrix.GeneralLinearGroup (Fin N) k))
    (hρ_alg : Etingof.IsAlgebraicRepresentation N M.ρ)
    (h_homog : -- weights all lie at degree n; see note below
       ∀ μ : Fin N →₀ ℕ, glWeightSpace k N M μ ≠ ⊥ → ∑ i, μ i = n) :
    ∃ (mult : {λ : Fin N → ℕ // Antitone λ ∧ ∑ i, λ i = n} → ℕ),
      Nonempty (M ≅ ⨁ (λ : {λ : Fin N → ℕ // Antitone λ ∧ ∑ i, λ i = n}),
        (Fin (mult λ) → SchurModule k N λ.val)) := sorry
```

Exact statement form is at the author's discretion — the important
shape is:

- **Polynomial + homogeneous of degree n** hypothesis, compatible with
  the API shape produced by sub-issue #2b (`#2478`).
- **Conclusion**: a (possibly non-canonical) multiplicity-indexed iso
  to a direct sum of concrete `SchurModule` objects.

The statement should be consumable by sub-issue #6 (this issue's
successor), which uses it via character matching — so exposing
multiplicities as a function `{antitone λ} → ℕ` (or equivalently a
`Finsupp`) is useful.

### Proof outline (from scoping doc §5 Step E)

1. **Embed `M` into `(V^⊗n)^m`** using `#2478`
   (`polynomialRep_embeds_in_tensorPower`) for some `m`.
2. **Decompose `(V^⊗n)^m` via Schur-Weyl** using
   `Theorem5_18_4_decomposition` applied to `V^⊗n` and summed over the
   `m` copies (or, equivalently, view `(V^⊗n)^m` as the tensor power
   with multiplicity):

   `(V^⊗n)^m ≅ ⨁_λ (Specht_λ × m copies) ⊗ SchurModule k N λ`

   using `#2458`'s `schurWeyl_gl_decomposition`.
3. **Restrict the decomposition to the subrep `M`**: since `M` is a
   GL_N-equivariant subrep and the ambient `(V^⊗n)^m` is semisimple
   (Mathlib `IsSemisimple` for GL_N-representations in characteristic
   zero, or from `Theorem5_18_4_semisimple`), `M` decomposes as a
   submodule of each summand. Because each `SchurModule k N λ` is
   irreducible (a known result — confirm via existing infrastructure
   or prove locally), the submodule intersection with each summand is
   a direct sum of copies of `SchurModule k N λ`.
4. **Bundle multiplicities**: let `mult λ := number of copies of
   SchurModule k N λ in M`. Combine into the claimed iso.
